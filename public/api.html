Pushing Live Updates to GuardMap

HTTP request: POST https://www.guard-map.herokuapp.com/api/push

Required Properties

  • time_created (UTC time, date object) : The time of incident
  • lat : The latitude of incident location
  • long : The longitude of incident location
  • severity : The severity ranking of the incident. See below table for more details

Severity Rankings

  • Shooting 10
  • Assault 7
  • Arson 5
  • Burglary 5
  • Robbery 5
  • Theft 5
  • Vandalism 3
  • Other 2
